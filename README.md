# DroneShowCreator
This Blender add-on allow to export 3D models into data readable in another add-on made in FlightSimulator. 

## How to use it
Don't use directly the Add-on into Blender, as recommended by Blender, create a symbolic link instead

Windows : mklink /D "C:\BlenderAddons\DroneShowCreator" "PathTo\DroneShowCreator"

Then go on Blender>Edit>Preferences>Get Extensions and go to the Repositories settings to add your local repository "C:\BlenderAddons\DroneShowCreator"
